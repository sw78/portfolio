O mnie

20+ lat w B2B / operacjach / wdrożeniach ERP, WMS i MES. Wszystkie narzędzia
w tym repozytorium powstały we współpracy z AI, żeby rozwiązać realne problemy

przedwdrożeniowej ERP.

📦 Projekty

Dashboard Produkcyjny

Panel monitorowania produkcji w czasie rzeczywistym — OEE, realizacja planu,
przestoje, koszty, WIP, traceability. Automatyczne alerty (np. wykrywanie
maszyny psującej się zbyt często).
Stack: HTML/JS, Chart.js, IndexedDB

Grafik Pro

System grafikowania zmianowego z automatyczną walidacją Kodeksu Pracy
(11h odpoczynku, 35h/tydzień, wykrywanie zbyt długich ciągów zmian bez wolnego)
i eksportem do Excela.
Stack: HTML/JS, localStorage

MPM-MFG — Process Mining

Odkrywanie procesów produkcyjnych z danych CSV — analiza wariantów, wąskich
gardeł, obciążenia zasobów.
Stack: HTML/JS, PapaParse, Chart.js

Wszystkie trzy powyższe mają wbudowany przycisk "🧪 Wczytaj dane demo" —
jedno kliknięcie i aplikacja pokazuje się w pełni wypełniona przykładowymi danymi,
bez potrzeby wskazywania żadnych plików.

System Analizy Przedwdrożeniowej ERP

Aplikacja desktopowa (Electron) dla konsultantów ERP — zastępuje rozproszone
arkusze Excel i dokumenty Word jednym ustrukturyzowanym narzędziem: 14 sekcji
analizy, wymagania FIT/GAP z priorytetami MoSCoW, harmonogram projektu (Gantt),
kosztorys.

Ta aplikacja działa inaczej niż pozostałe trzy — to nie strona internetowa,
tylko program do zainstalowania na Windows, dodatkowo zabezpieczony licencyjnie
(przypisanie do konkretnego komputera). Na stronie głównej portfolio, w karcie
tego projektu, znajdziesz plik instalatora do pobrania oraz przycisk kontaktowy
"Zapytaj o demo" — po instalacji aplikacja sama poprowadzi dalej (ekran aktywacji
wyświetli, jakie dane należy przesłać, żeby otrzymać licencję demonstracyjną).
